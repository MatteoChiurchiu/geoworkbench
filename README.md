# GeoWorkbench Deploy

Pacchetto pronto per pubblicazione su GitHub Pages.

## Struttura

- `GeoWorkbench/`
- `Build/CesiumDev/`
- `Source/Widgets/`

## Esecuzione locale rapida

Apri `GeoWorkbench/index.html` da un web server statico.

## Pubblicazione su GitHub Pages

1. Inizializza git nella cartella.
2. Pusha su una nuova repository GitHub.
3. In GitHub: Settings > Pages > Deploy from branch > `main` e `/ (root)`.
4. URL app: `https://<username>.github.io/<repo>/GeoWorkbench/`
